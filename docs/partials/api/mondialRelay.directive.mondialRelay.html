mondialRelay
directive in module mondialRelay

Description

To use mondialRelay, you had to inject mondial-relay as dependency and use it like this example 

 angular.module('myApp', ['mondial-relay]);

Usage

as element:
<mondial-relay
       ng-model="{string}">
</mondial-relay>
as attribute
<ANY mondial-relay
     ng-model="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ng-modelstring

Variable that will recieve the selected relay.
